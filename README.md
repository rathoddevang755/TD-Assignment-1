Python Basics: Assignment 1
Welcome to my first Python project! This repository contains a simple script designed to practice core programming concepts like user input, data type conversion, and string manipulation.

🚀 Overview
This project is divided into two primary tasks that demonstrate fundamental Python capabilities:

Basic Mathematical Operations: Handling numerical input and performing arithmetic.

Personalized Greeting: Collecting string data to interact with the user.

🛠 Features
Task 1: Calculator Logic
The script prompts the user for two numbers (floats) and performs the following:

Addition: Sums the numbers and rounds to the nearest whole number.

Subtraction: Calculates the difference and rounds the result.

Multiplication: Multiplies the numbers and rounds the result.

Division: Provides the precise quotient.

Task 2: String Formatting
A simple interaction module that:

Accepts a First Name and Last Name.

Combines them into a full name using string concatenation.

Outputs a customized welcome message.
